Lista de Comandos

  • ls: Lista os arquivos e diretórios no diretório atual.
  • cd: Navega entre diretórios.
  • pwd: Mostra o caminho completo do diretório atual.
  • mkdir: Cria um novo diretório.
  • rm: Remove arquivos ou diretórios (use com cuidado).
  • sudo: Executa comandos com privilégios de administrador.
  • apt-get update: Atualiza a lista de pacotes disponíveis.
  • apt-get upgrade: Atualiza todos os pacotes instalados para suas versões mais recentes.
  • top: Exibe os processos em execução e informações de uso do sistema.
  • df -h: Mostra o uso de espaço em disco de forma legível.
  • free -h: Exibe o uso de memória (RAM) do sistema.
  • nano: Abre o editor de texto Nano para editar arquivos diretamente no terminal.
  • cp: Copia arquivos ou diretórios.
  • mv: Move ou renomeia arquivos ou diretórios.
  • tar -czvf arquivo.tar.gz pasta/: Compacta a pasta em um arquivo `.tar.gz`.
  • tar -xzvf arquivo.tar.gz: Extrai um arquivo `.tar.gz`.
  • chmod: Altera as permissões de arquivos e diretórios.
  • chown: Altera o proprietário de arquivos e diretórios.
  • systemctl status [serviço]: Verifica o status de um serviço (exemplo: Apache ou MariaDB).
  • service [serviço] start: Inicia um serviço (exemplo: MariaDB).
  • service [serviço] stop: Para um serviço.
  • mysql -u root -p: Acessa o console do MariaDB/MySQL como administrador.
  • CREATE DATABASE nome_do_banco;: Cria um novo banco de dados no MariaDB.
  • SHOW DATABASES;: Lista todos os bancos de dados disponíveis no MariaDB.
  • DROP DATABASE nome_do_banco;: Exclui um banco de dados no MariaDB (use com cuidado).
  • exit: Sai do terminal ou de aplicativos como o console do MariaDB.
